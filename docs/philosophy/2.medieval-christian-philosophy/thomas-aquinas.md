# 托马斯·阿奎那 (Thomas Aquinas, 1225-1274)

## 主要思想

1. **信仰与理性的和谐**
   - 提出"信仰寻求理解"(Fides quaerens intellectum)
   - 认为理性与信仰最终都指向真理
   - 区分神学真理与哲学真理

2. **五路证明**
   - 第一推动者证明
   - 因果关系证明
   - 必然性证明
   - 完美性证明
   - 目的论证明

3. **自然法理论**
   - 永恒法：神的智慧
   - 自然法：人类理性对永恒法的参与
   - 人定法：具体社会中的法律
   - 神法：通过启示获得

4. **形而上学**
   - 存在与本质的区分
   - 潜能与现实理论
   - 四因说（质料因、形式因、动力因、目的因）

## 主要贡献

1. **神学大全**
   - 系统整合基督教教义与亚里士多德哲学
   - 建立完整的经院哲学体系
   - 影响天主教神学至今

2. **教育贡献**
   - 发展经院哲学的教学方法
   - 推动大学教育体系发展
   - 影响现代学术研究范式

3. **法律哲学**
   - 发展自然法传统
   - 影响后世法哲学思想
   - 为现代人权理论奠定基础

4. **伦理思想**
   - 德性伦理学的发展
   - 幸福论与至善论
   - 正义与共同善的理论

## 主要著作

1. **《神学大全》** (Summa Theologica)
   - 中世纪经院哲学的巅峰之作
   - 系统阐述神学、伦理学、政治哲学
   - 采用问答体写作

2. **《反异教大全》** (Summa contra Gentiles)
   - 为传教士写作的护教著作
   - 针对非基督徒的理性论证
   - 展示自然神学可能性

3. **《论存在与本质》** (De Ente et Essentia)
   - 早期形上学著作
   - 讨论存在与本质的关系
   - 奠定其形上学基础

4. **《论真理》** (De Veritate)
   - 讨论真理的本质
   - 包含129个神学问题
   - 展示其辩证方法

## 历史影响

1. **神学影响**
   - 成为天主教官方神学
   - 影响特伦特会议
   - 新托马斯主义复兴

2. **哲学影响**
   - 影响笛卡尔、莱布尼茨等近代哲学家
   - 对存在主义、现象学有启发
   - 当代分析哲学中的复兴

3. **文化影响**
   - 影响但丁《神曲》
   - 西方教育体系形成
   - 科学与宗教对话的桥梁
